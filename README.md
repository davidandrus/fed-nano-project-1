#Running locally:

**NOTE: you must have node installed**

`git clone git@github.com:davidandrus/fed-nano-project-1.git`

`cd fed-nano-project-1` or whichever folder you just created

`npm install`

`npm start`

in a browser navigate to `http://localhost:8080`
